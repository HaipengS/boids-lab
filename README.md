# Boids Lab

An interactive, web-based simulation of flocking behavior based on the
classic Boids model. This project explores how simple local interaction
rules give rise to complex emergent group behavior.

## Features
- Separation, alignment, and cohesion rules
- Real-time parameter tuning
- Interactive visualization using HTML5 Canvas
- Designed for extensibility (predators, wind, perching behavior)
